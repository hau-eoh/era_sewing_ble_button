Cấu Hình Kết Nối Trên E-Ra IoT Platform

Để hiển thị dữ liệu thực tế từ thiết bị phần cứng lên iFrame này, bạn hãy làm theo các bước hướng dẫn dưới đây.

1. Định dạng JSON Payload của Thiết bị

Mỗi thiết bị nút nhấn của công nhân cần gửi dữ liệu dạng chuỗi JSON về một Virtual Pin trên E-Ra với cấu trúc mẫu như sau:

{
  "node_id": 1,
  "event": "press",
  "press_count": 7,
  "battery_mv": 380,
  "battery_pct": 85,
  "rssi": -51,
  "online": true,
  "last_seen_epoch": 1782449719,
  "gateway_uptime_ms": 428777
}

2. Cấu hình Widget iFrame trên Dashboard E-Ra

Trong giao diện Dashboard của E-Ra, bạn thêm một Widget iFrame và trỏ URL đến trang web này (ví dụ trang đã được deploy lên GitHub Pages).

3. Liên kết Virtual Pin

Trong phần cài đặt Widget iFrame của E-Ra, liên kết datastream chứa Virtual Pin nhận tin nhắn JSON từ các nút nhấn máy may. SDK E-Ra Widget sẽ tự động bắt dữ liệu và cập nhật sơ đồ thời gian thực.
